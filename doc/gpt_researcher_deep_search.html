03 五大创新亮点
01

递归广度×深度搜索树

两个参数控制探索规模：breadth 决定每层子查询数，depth 决定递归层数。每层广度自动减半，实现先广后深的逐层聚焦。

breadth=4 depth=2 max(2, b//2)
02

LLM 驱动自主闭环

Strategic LLM 自动生成搜索查询和后续研究问题，形成"问题→搜索→提炼→新问题"的自主研究循环，无需人工干预。

Strategic LLM ReasoningEfforts.High
03

Semaphore 并发控制

asyncio.Semaphore 限制并发数（默认 2），每个子查询创建完整独立的 GPTResearcher 实例，兼顾执行效率与 API 稳定性。

asyncio.Semaphore concurrency=2
04

学习要点 + 引用溯源

从检索结果中提取结构化 learnings，每条关联源 URL 形成 citations 字典，确保研究结论可追溯、可验证。

learnings[] citations{}
05

上下文词量裁剪

trim_context_to_word_limit 从列表末尾（最新内容）倒序累加，保留最近最相关的内容，确保不超 25,000 词上限。

MAX_CONTEXT_WORDS reversed()
+

独立实例隔离

每个子查询实例化独立 GPTResearcher，各自维护 visited_urls / context / sources，避免状态污染，支持 MCP 配置透传。

GPTResearcher() mcp_configs
